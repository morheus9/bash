Обновить список пакетов
```
sudo pacman -Sy
```
Обновить все пакеты
```
sudo pacman -Syyu
```
Обновить все пакеты (+AUR)
```
yay -Syu
```
Установить пакет
```
sudo pacman -S foo
yay -S foo
```
Удалить пакет и зависимости
```
sudo pacman -Rns foo
yay -Rns foo
```
Найти пакеты AUR
```
sudo pacman -Qm
```
Найти ненужные зависимости
```
yay -Yc
```
Список из AUR:
```
visual-studio-code-bin
yandex-browser
```
